## 前缀和与差分

<!--more-->

### 前缀和

前缀和是一种重要的预处理技术，可以大大降低查询的时间复杂度。

#### 一维前缀和

对于数组 $a[1], a[2], ..., a[n]$，其前缀和数组 $s$ 定义为：

$$s[i] = \sum_{j=1}^{i} a[j] = a[1] + a[2] + ... + a[i]$$

通过前缀和，我们可以在 $O(1)$ 的时间内求出任意区间的和：

$$\sum_{j=l}^{r} a[j] = s[r] - s[l-1]$$

```cpp
// 一维前缀和
const int N = 100010;
int a[N], s[N];

// 预处理前缀和
for (int i = 1; i <= n; i++) {
    s[i] = s[i - 1] + a[i];
}

// 查询区间 [l, r] 的和
int query(int l, int r) {
    return s[r] - s[l - 1];
}
```

#### 二维前缀和

对于二维矩阵，前缀和的定义为：

$$s[i][j] = \sum_{x=1}^{i} \sum_{y=1}^{j} a[x][y]$$

查询子矩阵 $(x1, y1)$ 到 $(x2, y2)$ 的和：

$$sum = s[x2][y2] - s[x1-1][y2] - s[x2][y1-1] + s[x1-1][y1-1]$$

```cpp
// 二维前缀和
const int N = 1010;
int a[N][N], s[N][N];

// 预处理
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        s[i][j] = s[i-1][j] + s[i][j-1] - s[i-1][j-1] + a[i][j];
    }
}

// 查询
int query(int x1, int y1, int x2, int y2) {
    return s[x2][y2] - s[x1-1][y2] - s[x2][y1-1] + s[x1-1][y1-1];
}
```

### 差分

差分是前缀和的逆运算，常用于区间修改问题。

#### 一维差分

对于数组 $a$，其差分数组 $b$ 满足：

$$a[i] = \sum_{j=1}^{i} b[j]$$

即 $a$ 是 $b$ 的前缀和。

**区间修改**：将区间 $[l, r]$ 内的所有数加上 $c$

```cpp
// 差分数组操作
b[l] += c;
b[r + 1] -= c;

// 最后求前缀和恢复原数组
for (int i = 1; i <= n; i++) {
    a[i] = a[i-1] + b[i];
}
```

#### 二维差分

对于二维矩阵的区间修改（将子矩阵 $(x1, y1)$ 到 $(x2, y2)$ 的所有元素加上 $c$）：

```cpp
// 二维差分操作
b[x1][y1] += c;
b[x1][y2+1] -= c;
b[x2+1][y1] -= c;
b[x2+1][y2+1] += c;
```

### 应用场景

- **前缀和**：频繁查询区间和、区间平均值等
- **差分**：区间统一加减操作、离线区间修改问题

### 时间复杂度

- 预处理：$O(n)$
- 单次查询/修改：$O(1)$
- 总体复杂度：$O(n + q)$，其中 $q$ 为查询次数
