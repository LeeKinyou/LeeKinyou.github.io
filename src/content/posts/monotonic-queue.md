## 单调队列

> 这里引用百度的解释：
>

<hr>

### 1. 从例题引入

POJ 2823 [Sliding Window](http://poj.org/problem?id=2823)

题意：给出一个长度为 $n$ 的数组，输出每 $k$ 个连续的数中的最大值和最小值。

<!--more-->

- 暴力的想法是直接寻找 $n-k+1$ 个区间，分别求每个区间的最大值最小值。显然这个办法是不行的，时间复杂度为 $O(n \times k)$

- 那么如何优化，显然这其中进行了大量重复工作，除了开头 $k-1$ 个和结尾 $k-1$ 个数之外，每个数都进行了 $k$ 次比较，这样是极其不划算的，这里引出单调队列的做法。

<hr>

### 2. 单调队列详解

*单调队列* 是队列内元素满足单调性的队列结构。<font color=Red>且为了满足队列内元素的单调性，队尾也可弹出元素。</font>此处的单调性分为单调递增与单调递减，为了便于描述，接下来以「单调递增队列」为例进行讲解。

<b>单调递增队列中 *队尾* 的操作与 单调递增栈中 *栈顶* 的操作一致，即假设当前想要入队的元素为 $x$，若队尾元素 $a_i \le x$，则将 $x$ 入队，否则不断弹出队尾元素，直至队尾元素 $a_i \le x$。</b>

例如以数组 1 4 3 5 2 7 为例，若「队首」始终不弹出元素，则其具体过程如下图所示。

| 情况如下                                                     |
| ------------------------------------------------------------ |
| $1$ 入队                                               -> *[1]* |
| $4 \ge 1$, $4$ 入队                               -> *[1, 4]*    |
| $3 < 4$, $4$ 出队;  $3 \ge 1$, $3$ 入队      -> *[1, 3]*            |
| $5 \ge 3$, $5$ 入队                               -> *[1, 3, 5]* |
| ...                                                          |
| 最后的队列为 *[1, 2, 7]*                                     |

由此可知，<font color=Red>单调队列与单调栈的最大区别就在于 *队首* 的操作，何时将队首元素 pop 是单调队列算法的关键。</font>

然而队首的操作往往具有多样性，并非一成不变，需要根据情况而定。

我们再来看看例题：

要求的是每连续的 $k$ 个数中的最大（最小）值，很明显，当一个数进入所要 "寻找" 最大值的范围中时，若这个数比其前面（先进队）的数要大，显然，前面的数会比这个数先出队且不再可能是最大值。

也就是说——当满足以上条件时，可将前面的数 "弹出"，再将该数真正 push 进队尾。

这就相当于维护了一个递减的队列，符合单调队列的定义，减少了重复的比较次数，不仅如此，由于维护出的队伍是查询范围内的且是递减的，队头必定是该查询区域内的最大值，因此输出时只需输出队头即可。

显而易见的是，在这样的算法中，每个数只要进队与出队各一次，因此时间复杂度被降到了 $O(n)$。

而由于查询区间长度是固定的，超出查询空间的值再大也不能输出，因此还需要 site 数组记录第 $i$ 个队中的数在原数组中的位置，以弹出越界的队头。

### 3. 代码实现

```cpp
#include <iostream>
#include <deque>
using namespace std;

const int N = 1000010;
int a[N];
deque<int> q_min, q_max;

int main() {
    int n, k;
    cin >> n >> k;
    
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    // 求滑动窗口最小值
    for (int i = 0; i < n; i++) {
        // 弹出队首过期元素
        while (!q_min.empty() && q_min.front() <= i - k) {
            q_min.pop_front();
        }
        // 维护单调递增队列
        while (!q_min.empty() && a[q_min.back()] >= a[i]) {
            q_min.pop_back();
        }
        q_min.push_back(i);
        
        if (i >= k - 1) {
            cout << a[q_min.front()] << " ";
        }
    }
    cout << endl;
    
    // 求滑动窗口最大值
    for (int i = 0; i < n; i++) {
        while (!q_max.empty() && q_max.front() <= i - k) {
            q_max.pop_front();
        }
        while (!q_max.empty() && a[q_max.back()] <= a[i]) {
            q_max.pop_back();
        }
        q_max.push_back(i);
        
        if (i >= k - 1) {
            cout << a[q_max.front()] << " ";
        }
    }
    
    return 0;
}
```

### 4. 总结

单调队列的核心思想：

1. **维护单调性**：队尾插入时，弹出破坏单调性的元素
2. **维护窗口**：队首弹出时，移除超出窗口范围的元素
3. **获取答案**：队首元素即为当前窗口的最优值

时间复杂度：$O(n)$，每个元素最多入队和出队各一次。
